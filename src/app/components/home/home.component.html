Help Desk Programando para trabalho

Olá! Seja bem vindos ao sistema Help Desk desenvolvido por mim no curso Desenvolvedor Full Stack.
Este é um sistema para gerenciamento de chamados simulando um simples Help Desk desenvolvido em Angular 12 e Spring Boot 2.
O sistema está rodando em nuvem, sendo o Front da Vercel e o Back no Heroku.
O projeto tem como principal objetivo abordar os conceitos de aplicações SPAs, Autenticação e Autorização com Token JWT e muito mais.
